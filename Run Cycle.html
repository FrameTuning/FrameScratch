Scratch On! The Site is still in work in progress ( WIP ).

To make a good Run cycle you've gotta be able to understand how to walk & how to animate (optional)

For this tutorial we are going to need somethings.
1. You have to be able to understand how to use Scratch
2. You gotta have a Sprite with a Run Cycle
3. That's it.


Let's get coding!


Let's begin with creating the character!
For this tutorial, I'm using Griffpatch Scratch Cat Run Cycle.

For this tutorial, I'm gonna make the Cat Float and I'm only gonna focus on the Animation part. The Run Cycle.


Let's continue by making a new variable called "frame" for this sprite only.


Then create a simple script for the Left and Right Movement.

This is my own script feel free to copy it, it's just that you will not be able to move. You will be floating instead of moving.


Now, we want to switch the costume to the correct ones for our Run Cycle!


Let's look at which ones they are.
For me the run cycle begins at costume 1 and ends all the way down at costume number 16.
We will need those numbers (from 1 to 16).


Make a new Custom block and call it "Set Costume", DON'T make it Run without Screen Refresh.


After that put a Switch Costume Block under the Set Costume Custom Block.
Add a ( + ) sign on the Switch Costume Block with the first input as 1.


Then add the Set Costume custom block under the Tick - Player Broadcast Message.

For this tutorial, I'm gonna add a Set Frame variable to ( 0 ) when the Green Flag is Clicked

If you have the same script as me and you test the project, If you press both A and D, you will see that the Animation is faster. We will get to that later.


If you have frame other than the Run Cycle, this is for you.

In fact this is pretty simple to do.
Take a MOD operator and put it on the Second Input of the ( + ), then put the frame variable on the Left Input of the MOD operator, and on the Right Input, put 16 in it.


If you want to make sure there is no glitch in the Animation you can put a FLOOR operator on the Second Input of the ( + ), then putting the MOD operator on the Input of the FLOOR operator.

Remeber I said I will fix the A & D key bug, that's what I'm gonna do!


To fix this bug you will need to do TWO simple things.First Delete the If key A pressed script and put an OR operator on the If D pressed script, then put a key A pressed on the other Input of the OR operator.


It should now look like this.


To finish this tutorial, I'm gonna make the Idle Standing Pose.


To make the Idle Standing Pose, put an If else on the Switch Costume block, and on the Input, put a GREATER operator and on the Left input of the GREATER operator, put a Frame varible, and on the Right input a 0

After that, put a Switch Costume to *name-of-your-idle-pose on the ELSE.


With All of this, your Code should look like this.
